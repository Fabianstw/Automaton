## Automaton

Interactive React/Vite tool to build, visualize, and check Büchi automata against ω-words.

Website: https://automaton-pearl.vercel.app/

### Run locally
- Install deps: `bun install`
- Dev server: `bun run dev`
- Optional preview (after build): `bun run build && bun run preview`

### Format and test
- Format: `bun run format`
- Tests (lint + unit): `bun run test`
- Unit-only: `bun run test:unit`
